# Pulseboard S3
This is an ESP32 S3 based Devboard, designed to be used in conjunction with EMGs and EKGs. As with all ESP32S3 Devboards, it has WiFi and Bluetooth capibilities.
<img width="1111" height="711" alt="image" src="https://github.com/user-attachments/assets/68009f92-13fb-4497-8127-46e2f7b7dd58" />

## Features
* ESP32 S3 Microcontroller
* Wifi and Bluetooth
* Onboard Battery Charging
* Onboard temperature and moisture sensor
* 38 GPIO pins
* 3.3 and 5v
* USB-C Connectivity (Impedence matched)
* Status LEDS
* 4-layer PCB design
